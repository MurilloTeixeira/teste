instalar node e npm: https://nodejs.org/en/download/package-manager
instalar VueJs : https://br.vuejs.org/v2/guide/installation


Criar um projeto: https://cli.vuejs.org/guide/creating-a-project.html
    prompt$   vue create hello-world
OU por modo visual
    prompt$   vue ui 

Instalar Axios :  https://www.npmjs.com/package/axios
    npm i axios

Instalar Plugin Router VueJS: https://router.vuejs.org/installation
    npm i vue-router@4


para executar no codespace do git :
    1 - abra o arquivo package.json 
    2 - encontre a linha `"serve": "vue-cli-service serve",`
    3 - altere para `"serve": "vue-cli-service serve --mode production",`


para executar:
    $  npm run serve
